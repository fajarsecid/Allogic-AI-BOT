# 🤖 Allogic AI BOT

**Allogic AI BOT** adalah bot WhatsApp berbasis **Baileys** dengan fitur manajemen grup, AI assistant, dan smart command router.

Bot ini dirancang agar pengguna bisa berinteraksi secara natural dengan AI, lalu AI akan memahami maksud pesan dan menjalankan command bot yang sesuai secara otomatis.

---

## ✨ Fitur Utama

- ✅ **AI Chat / Assistant**
- ✅ **Smart Router** untuk mengubah perintah natural menjadi command bot
- ✅ **Mode publik / private**
- ✅ **Manajemen grup**
  - promote / demote admin
  - mute / unmute grup
  - tag all
  - welcome
  - anti link
  - antitag
  - ban / unban
- ✅ **Sticker tools**
- ✅ **Text maker**
- ✅ **Downloader & tools lain** (sesuai command yang tersedia di bot)
- ✅ **Support multi provider AI**
  - Google / Gemini
  - Groq
  - OpenRouter
- ✅ **Auto provider selection**
  - tugas ringan → provider ringan
  - tugas berat / thinking panjang → provider lebih kuat
